# saturate 0.1.0

* Initial CRAN release.
* DuckDB-backed project files (`.duckdb` / `.satdb`) — one portable file per project.
* Interactive Shiny GUI for text coding, codebook management, and multi-coder workflows.
* Hierarchical codebook with categories, code relations, weights, and full audit history.
* Split/merge workflow for independent blind coding and result reconciliation.
* Inter-rater reliability: Cohen's kappa, Krippendorff's alpha, agreement matrices.
* Saturation curves, triangulation grids, and temporal analysis.
* Member-checking: generate and track participant feedback on interpreted themes.
* Export to DOCX, HTML, plain text, XLSX, and tidy tibbles.
