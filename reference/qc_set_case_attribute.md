# Set a case attribute value (upsert)

Set a case attribute value (upsert)

## Usage

``` r
qc_set_case_attribute(project, case_id, variable, value)
```

## Arguments

- project:

  A `qc_project` object.

- case_id:

  Integer.

- variable:

  Character. Attribute name (e.g. `"age_group"`).

- value:

  Character.

## Value

Invisibly, `TRUE`.
