# Update the confidence score on a single coding

Update the confidence score on a single coding

## Usage

``` r
qc_update_coding_confidence(project, id, confidence)
```

## Arguments

- project:

  A `qc_project` object.

- id:

  Integer. Coding id.

- confidence:

  Integer 0-100 or `NULL` (unrated).

## Value

Invisibly `TRUE`.
