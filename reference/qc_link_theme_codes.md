# Link codes to a theme

Re-linking a previously unlinked code restores the association without
creating a duplicate row.

## Usage

``` r
qc_link_theme_codes(project, theme_id, code_ids)
```

## Arguments

- project:

  A `qc_project` object.

- theme_id:

  Integer. Theme id.

- code_ids:

  Integer vector. One or more code ids.

## Value

Invisibly `NULL`.
