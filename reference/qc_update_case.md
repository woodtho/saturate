# Update a case name and/or memo

Update a case name and/or memo

## Usage

``` r
qc_update_case(project, case_id, name = NULL, memo = NULL)
```

## Arguments

- project:

  A `qc_project` object.

- case_id:

  Integer.

- name:

  Character. New name (or `NULL` to leave unchanged).

- memo:

  Character. New memo (or `NULL` to leave unchanged).

## Value

Invisibly, `TRUE`.
