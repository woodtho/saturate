# Unlink a category from a theme

Unlink a category from a theme

## Usage

``` r
qc_unlink_theme_category(project, theme_id, category_id)
```

## Arguments

- project:

  A `qc_project` object.

- theme_id:

  Integer. Theme id.

- category_id:

  Integer. Category id.

## Value

Invisibly `NULL`.
